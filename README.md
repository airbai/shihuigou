时惠购
=========

#”时惠购“是一个Chrome 扩展。
使用"时惠购”，你可以关注/收藏 “京东商城(360buy.com), 天猫（tmall.com)”等中国电子商务网站商品的价格变化趋势。
同时可以根据设定条件（比如，降价5%）提醒你去购买。
